# 二郎記録 PoC

GitHub Pagesでそのまま公開できる、二郎記録の静的WebアプリPoCです。画像アップロード・各種トッピングの段階入力・一覧/詳細/編集/削除がローカルで完結します。

## ローカル起動方法

- `index.html` をそのままブラウザで開くだけでも動作します。
- 推奨: VSCodeのLive Serverなどで `index.html` を開くと、ハッシュルーティングの挙動確認がしやすいです。

## GitHub Pagesで公開する手順

1. GitHubの対象リポジトリで **Settings → Pages** を開く
2. **Branch** を `main`、**/root** を選択して保存
3. 数分待つと公開URLが発行されます

## 注意事項

- 画像は `localStorage` にDataURLで保存されます。容量制限に注意してください。
- 保存前に画像を縮小・圧縮しますが、大きな画像を大量に保存すると容量に達する場合があります。
